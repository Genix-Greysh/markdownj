MarkdownJ

A Java port of Markdown, the text-to-html conversion tool

This project is maintained by myabc

MarkdownJ is a pure Java port of John Gruber's incredible Markdown text-to-html conversion utility.

Where we’re going, we don’t need tags

Dependencies

JDK 1.5

What’s New?

See the latest release.

Install

with Maven: 
<dependency>
  <groupId>org.markdownj</groupId>
  <artifactId>markdownj</artifactId>
  <version>0.4.0</version>
</dependency>
with Gradle: 
  'org.markdownj:markdownj:0.4.0'

How do I use it?


  MarkdownProcessor markdown = new MarkdownProcessor();
  String html = markdown.markdown("*italic*   **bold**\n_italic_   __bold__");
