markdownj by myabc

A Java port of Markdown, the text-to-html conversion tool

MarkdownJ is the pure Java port of Markdown (a text-to-html conversion tool written by John Gruber.)

Dependencies

JDK 1.4

Install

See http://code.google.com/p/markdownj/wiki/Maven
